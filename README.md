# Simple WebSocket Echo Server

This example demonstrates a basic WebSocket echo server using Python's asyncio and websockets library. It listens for incoming connections, receives messages from clients, and sends the same messages back, illustrating the fundamental real-time communication concept.

## Language

`python`

## How to Run

1. Save the code as `websocket_server.py`.
2. Run it from your terminal: `python websocket_server.py`.
3. Connect using a WebSocket client (e.g., a browser's developer console or another script).
